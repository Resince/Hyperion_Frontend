# README

这是商场系统的前端部分,后端部分请移步[后端](https://github.com/duanyue166/Hyperion_Backend)

### 相关技术栈

+ vue3 使用组合式api
+ TypeScript
+ vite 作为构建工具
+ element-ui plus
+ tailwind css (较少)

### 功能
+ 用户端
    + 商品的浏览
    + 加入购物车
    + 购买商品
    + 订单
+ 商家端
    + 商品管理
    + 订单管理
+ 管理端
    + 用户管理

### 开始

#### 环境配置

需要 `node` 版本大于 `v20.11.1`

#### 启动

项目目录下
```
yarn i
yarn vite
```
